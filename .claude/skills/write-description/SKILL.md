# Write Description

Generate optimized eBay listing copy — title, description, category selection, item specifics, and listing format. This skill covers all supported collectible categories.

---

## Title Optimization

eBay titles have an **80-character limit**. Every character counts — titles are the primary search surface.

### General title rules
- Front-load the most searched terms (card name, set name, character name)
- Include condition/grade if graded (e.g., "PSA 10")
- Include variant details (foil, holo, chase, exclusive, 1st edition)
- Use common abbreviations buyers search for: NM, LP, PSA, BGS, CGC, SEALED, OOP
- NO special characters, emojis, or ALL CAPS words (eBay may demote)
- NO filler words: "LOOK!", "WOW!", "HOT!", "RARE!" (unless it genuinely is rare — then "Rare" is acceptable)
- Spell out AND abbreviate when space allows: "Pokemon" not "Pkmn" (but "MTG" is fine — everyone searches that)

### Title formulas by category

**Trading card single (raw):**
```
[Card Name] [Set Name] [Set Code/Number] [Variant] [Condition] [Language]
```
Example: `Sheoldred the Apocalypse Dominaria United DMU-107 NM Magic The Gathering MTG`

**Trading card single (graded):**
```
[Grade Company] [Grade] [Card Name] [Set Name] [Year] [Variant]
```
Example: `PSA 10 Charizard VMAX Shining Fates SV107 Full Art Pokemon 2021`

**Sealed product:**
```
[Product Type] [Set Name] [Game] [Edition] SEALED [Year]
```
Example: `Booster Box Prismatic Evolutions Pokemon TCG SEALED 2025 36 Packs`

**Funko Pop:**
```
Funko Pop [Character] #[Number] [Variant/Exclusive] [Line] [Condition Note]
```
Example: `Funko Pop Batman #01 Vaulted DC Heroes Vinyl Figure Mint Box`

**Disney pin:**
```
Disney Pin [Character/Design] [Series] [Edition Type] [LE Count] [Year]
```
Example: `Disney Pin Stitch Experiments Series Limited Edition LE/500 2019`

## Description Templates

### Trading Card Single

```
[CARD NAME] — [SET NAME] ([SET CODE] #[NUMBER])

Condition: [NM/LP/MP/HP/DMG] (or [GRADING COMPANY] [GRADE], Cert #[NUMBER])
Variant: [Regular / Foil / Full Art / Extended Art / etc.]
Edition: [1st Edition / Unlimited / Shadowless / N/A]
Language: [English / Japanese / etc.]

Card Details:
- [Rarity level]
- [Notable gameplay or collector significance]
- [Any distinguishing features — centering, print quality, notable flaws if LP+]

Condition Notes:
[Honest, specific description of condition. For raw cards: describe corners, edges, surface, centering. For graded: reference the grade and any subgrades.]

What You Get:
- [1x card as pictured]
- [Toploader + penny sleeve for shipping protection]
- [If graded: shipped in original slab]
```

### Sealed Card Product

```
[PRODUCT TYPE] — [SET NAME] ([GAME])

Contents: [Number of packs] packs, [cards per pack] cards per pack
Edition: [1st Edition / Unlimited / Standard]
Seal: [Factory sealed with [manufacturer] shrink wrap / Distributor sealed]
Language: [English / Japanese / etc.]

Product Details:
- [Set highlights — chase cards, notable reprints, set significance]
- [Product-specific contents if applicable — promo cards, dice, sleeves in ETBs]

Seal Condition:
[Describe shrink wrap condition — intact, tight, any tears or looseness. Note manufacturer seal/logo if visible.]

What You Get:
- [1x sealed product as pictured]
- [Shipped in a box with padding to prevent crushing]
```

### Funko Pop

```
Funko Pop! [CHARACTER] #[NUMBER] — [LINE/FRANCHISE]

Variant: [Common / Chase / Exclusive / Flocked / GITD / etc.]
Exclusive: [Store/Convention name, or N/A]
Vaulted: [Yes/No]
Box Condition: [Mint / Near Mint / Very Good / etc.]

Details:
- [Notable features — glow, flocked, metallic, etc.]
- [Exclusive sticker present: Yes/No — describe sticker]
- [Any paint or figure defects beyond normal QC]

Box Condition Notes:
[Specific description — corner condition, window clarity, any creases, dents, or shelf wear. This matters enormously for Funko buyers.]

What You Get:
- [1x Funko Pop in box as pictured]
- [Shipped in a Pop protector with bubble wrap]
```

### Disney Pin

```
Disney [PIN NAME/DESIGN] Pin

Edition: [LE/[count] / Open Edition / Hidden Mickey / etc.]
Series: [Series or collection name]
Year: [Release year]
Back Stamp: [Describe markings on back]

[Describe pin face, enamel quality, any scratches or wear. Note back stamp clarity and pin post condition.]

```

## Condition Disclosure Language

Be honest and specific. eBay's money-back guarantee means understating condition leads to returns.

**Good phrases:**
- "Please see all photos for exact condition"
- "Corners show minor whitening visible in photo 3"
- "Factory paint variation on [location] — not a defect, standard Funko QC"
- "Box has a small crease on [location] as shown in photos"
- "Card has been stored in a penny sleeve and toploader since pull"

**Avoid:**
- "Mint" unless it genuinely is (most raw cards are NM at best)
- "Pack fresh" — subjective and leads to disputes
- "Perfect" — nothing is perfect under magnification
- Hiding flaws behind "please see photos" without describing them

## eBay Categories and Item Specifics

### Trading Cards
- **MTG singles**: Collectible Card Games > Magic: The Gathering > Individual Cards
  - Item specifics: Game, Set, Rarity, Card Name, Card Number, Language, Card Condition, Foil/Non-Foil
- **Pokemon singles**: Collectible Card Games > Pokemon TCG > Individual Cards
  - Item specifics: Set, Card Name, Card Number, Rarity, Graded/Ungraded, Language, Card Condition
- **Sports singles**: Sports Trading Cards > [Sport] > Trading Cards
  - Item specifics: Sport, Player, Team, Year, Set, Card Manufacturer, Card Number, Graded/Ungraded

### Sealed Products
- **MTG sealed**: Collectible Card Games > Magic: The Gathering > Sealed Booster Packs/Boxes
- **Pokemon sealed**: Collectible Card Games > Pokemon TCG > Sealed Booster Packs/Boxes
- **Sports sealed**: Sports Trading Cards > [Sport] > Sealed Boxes/Cases

### Funko Pops
- Action Figures & Accessories > Action Figures > Funko
  - Item specifics: Brand (Funko), Character, Franchise, Type (Pop!), Size, Features (Chase, Exclusive, Flocked, etc.)

### Disney Pins
- Collectibles > Disneyana > Contemporary (1968-Now) > Pins, Patches & Buttons > Pins
  - Item specifics: Brand (Disney), Theme, Character, Type (Pin), Era

## Listing Format Guidance

### Auction vs Buy It Now

**Use Auction when:**
- You're unsure of market value and want price discovery
- The item is rare/unique with unpredictable demand (1/1 cards, error prints)
- Comps are sparse or highly variable
- You want to move the item quickly (7-day auction)

**Use Buy It Now (BIN) when:**
- Comps are consistent and you know the market price
- The item is a commodity with regular sales (in-print sealed product, common Pops)
- You want to maximize price and can wait for the right buyer
- You're listing graded cards with specific cert numbers (buyers search by grade)

**BIN + Best Offer:**
- Best default for most collectibles
- Set your BIN at the top of recent comps
- Auto-accept offers at or above your target price
- Auto-decline offers below your floor (saves time)

### Scheduling and timing
- **Best end times**: Sunday 6–9 PM local time (highest traffic for collectibles)
- **For auctions**: Start 7 days before a good end time
- **Avoid**: Ending during weekday work hours, holidays, or major competing events
- **Promoted listings**: 2–5% ad rate is usually sufficient for collectibles. Higher for competitive categories

### Lot bundling
- Bundle commons/low-value items into themed lots ("MTG Commander Staples Lot — 50 Cards")
- Don't bundle valuable items with junk — the valuable item gets diluted
- Lot titles should emphasize total value or theme, not individual cards
- Include a full list of contents in the description

## Standard Boilerplate

### Shipping terms
```
Shipping: [Method] with tracking. Ships within [1-2] business days of payment.
Combined shipping available — message before purchasing for a quote.
International buyers: please message for shipping quote before purchasing.
```

### Return policy
```
Returns accepted within 30 days. Item must be returned in the same condition as sent.
Buyer pays return shipping unless item was not as described.
```

### Disclaimer (for raw cards)
```
Condition is assessed to the best of my ability. Please review all photos carefully.
I am not a professional grader — if condition is critical, please ask questions before purchasing.
```
